# xash3d-api-server

`xash3d-api-server` provides FFI definitions necessary to interoperate with
[Xash3D engine](https://github.com/FWGS/xash3d-fwgs) in a server DLL.

# Usage

Add the following to your `Cargo.toml`:

```toml
[dependencies]
xash3d-api = "0.2"
xash3d-api-server = "0.1"
```

# Minimum Supported Rust Version (MSRV)

This version of crate requires Rust `1.68` or later.
